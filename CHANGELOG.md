## 1.0.0

* Initial release
* Phone number validation for Indonesian formats (08xx, +62, 62)
* NIK (Nomor Induk Kependudukan) validation with optional region checking
* NPWP (Nomor Pokok Wajib Pajak) validation with formatting support
* Email validation (RFC 5322 compliant)
* Bilingual support (Indonesian and English error messages)
* Validator chaining support
* Custom validator creation
* Strict validation mode
* Full null safety
* 100% test coverage
